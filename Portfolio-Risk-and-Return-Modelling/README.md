This folder contains portfolio construction and risk analysis models built using Microsoft Excel.

The work includes:

- Fundamental ratio analysis of selected NIFTY 50 companies  
- Calculation of daily returns and annualized volatility  
- Markowitz Mean-Variance portfolio optimization  
- Covariance matrix computation  
- Single Index Model (SIM) regression analysis  
- Beta, Alpha, and R-squared estimation  
- Portfolio variance and systematic risk comparison  

These models demonstrate practical application of modern portfolio theory and quantitative risk-return evaluation.
