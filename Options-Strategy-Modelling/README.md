This project demonstrates the construction and evaluation of an options trading strategy using Microsoft Excel.

The model includes:

- Bear Put Spread strategy implementation  
- Net premium calculation  
- Profit and Loss computation  
- Breakeven analysis  
- Payoff diagram construction  
- Basic Greeks interpretation (Delta, Theta, Gamma, Vega, Rho)

The objective is to understand structured options strategies, risk-reward profiles, and payoff behavior under different market scenarios.
