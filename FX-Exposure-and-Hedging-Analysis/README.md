This project analyzes foreign exchange exposure and hedging strategies using real company data.

The model includes:

- Identification of transaction exposure  
- Export revenue sensitivity to exchange rate movements  
- FX impact analysis on financial performance  
- Hedging strategy evaluation  
- Scenario and sensitivity analysis  
- Related party transaction assessment  

The objective is to understand how exchange rate fluctuations affect corporate earnings and how hedging tools can be used to manage currency risk.
