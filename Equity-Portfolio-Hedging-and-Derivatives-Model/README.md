This project demonstrates practical hedging of an equity portfolio using Futures and Options.

The model includes:

- Portfolio exposure calculation  
- Futures-based hedge implementation  
- Protective Put (Options) hedge modelling  
- Breakeven point computation  
- Scenario-based spot price simulation  
- Profit and Loss comparison (Hedged vs Unhedged)  
- Dynamic payoff diagram construction  

The objective of this model is to evaluate risk reduction techniques and compare hedging effectiveness using derivative instruments.
