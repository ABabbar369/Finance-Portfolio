This project demonstrates an ESG-integrated Discounted Cash Flow (DCF) valuation model built using Microsoft Excel.

The model includes:

- Free Cash Flow to Firm (FCFF) computation  
- WACC estimation  
- CAPM-based cost of equity  
- Terminal value calculation  
- Sensitivity analysis  
- ESG-adjusted cash flow impact assessment  

The objective is to evaluate how sustainability factors influence intrinsic valuation and long-term financial performance.
