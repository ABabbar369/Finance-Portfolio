This folder contains applied financial modeling and valuation projects built using Microsoft Excel.

The work includes:

- Integrated three-statement financial modeling (Income Statement, Balance Sheet, Cash Flow)
- Ratio analysis and DuPont decomposition
- Discounted Cash Flow (DCF) valuation
- CAPM-based cost of equity estimation
- Terminal value calculation
- Capital growth and SIP simulation modeling
- Sensitivity and scenario analysis

All models are dynamically linked, structured for clarity, and designed to demonstrate practical valuation, financial analysis, and modeling skills.
